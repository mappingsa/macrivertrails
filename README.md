rivertrails
===========

Mapping Services Australia rivertrails

This is a refactor of the Mapping Services Australia rivertrails site using Middleman static
site generator.

Prerequisites
-------------
Ruby 1.8.1 -> 1.9.3
RubyGems
Middleman 3.0.5
Slim 1.3.0 (Do not use Slim 1.3.1 - not compatible with Middleman)
